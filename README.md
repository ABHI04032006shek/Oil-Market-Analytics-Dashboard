# 🛢️ Oil Market Analytics Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive **Oil Market Analytics Dashboard** built using Power BI. It provides insights into oil production, pricing trends, and regional performance to support data-driven decision-making in the energy sector.

---

## 🎯 Objectives

* Analyze oil production and consumption trends
* Track oil price fluctuations over time
* Compare regional performance and output
* Provide actionable insights for business and economic analysis

---

## 📊 Key Features

* 📈 Time-series analysis of oil prices
* 🌍 Region-wise production comparison
* 💰 Revenue and pricing insights
* 📅 Yearly and monthly trend analysis
* 🎛️ Interactive filters and slicers for dynamic exploration

---

## 🧰 Tools & Technologies

* **Power BI Desktop (.pbix)**
* Data Modeling & Relationships
* DAX (Data Analysis Expressions)
* Data Visualization Techniques

---

## 📂 Project File

* `oil.dashboard.pbix` → Main dashboard file

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open in **Power BI Desktop**
3. Use slicers and filters to explore insights

---

## 💡 Key Insights

* Identified trends in oil price fluctuations over time
* Compared production output across different regions
* Highlighted patterns in supply and demand
